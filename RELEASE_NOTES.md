# PS5-MemoryPeeker v1.2Beta

Initial public release.

- PS5Debug/libdebug network connection.
- EBOOT process hook for offline game memory editing.
- Process memory section refresh and filtering.
- First scan and next scan value narrowing.
- Address read/write with locked value support.
- Cheat table save/load.
- JSON, SHN, and MC4 export.
